## 📌 Project Description

This project implements a **hardware-based authentication system** using biometric (fingerprint) and NFC to securely unlock an encrypted NVMe SSD without manual password input.

The system is designed to address common security risks in traditional password-based disk encryption, such as brute-force attacks and credential leakage, by introducing **multi-factor authentication with external hardware validation**.

Built on **Arduino Mega 2560**, the system integrates:

* Fingerprint sensor (AS608)
* NFC module (PN532)
* External EEPROM for secure data storage
* Real-Time Clock (RTC) for timestamp validation
* Custom power management module

During operation, the user authenticates via fingerprint or NFC. Upon successful verification, the Arduino sends an **encrypted authentication key** to a Windows system, which then decrypts the NVMe SSD using VeraCrypt.

The system also includes:

* 🔋 Portable power design (Li-ion + TP4056 + boost converter)
* ⚡ Standby power control using relay (to reduce attack surface & power usage)
* 🔐 Secure communication with encryption & validation mechanisms

### 🚀 Performance

* Fingerprint authentication: ~0.74 – 0.84 seconds
* NFC authentication: ~0.74 – 0.94 seconds
* SSD unlock process: ~2.95 – 7.82 seconds

These results demonstrate that hardware-based authentication can effectively replace manual password entry while maintaining **security, efficiency, and usability**.

### 🎯 Key Contribution

This project provides a **practical embedded security solution** combining:

* Hardware authentication
* Cryptographic protection
* Secure storage access control

It is suitable for applications in **embedded systems, IoT security, and data protection systems**.
